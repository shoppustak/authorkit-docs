# Managing Books

## Introduction to Book Management

AuthorKit for Books provides a complete system for managing your book catalog directly within WordPress. Whether you're adding your first book or managing an extensive library, the interface is intuitive and designed for authors, not developers.

## Getting Started

Managing books in AuthorKit works just like managing WordPress posts or pages. If you're familiar with WordPress, you'll feel right at home.

### Accessing Book Management

From your WordPress admin dashboard:

1. Log in to WordPress admin
2. Look for **Books** in the left sidebar menu
3. Click to expand and access:
   - **All Books** - View and manage existing books
   - **Add New** - Create a new book
   - **Categories** - Manage genres and categories
   - **Series** - Organize books into series
   - **Age Ranges** - Tag books by reader age

## Organizing Your Catalog

### Categories (Genres)

Organize books by genre using WordPress taxonomies. Create categories like Fantasy, Romance, Mystery, Science Fiction, and more. Books can belong to multiple categories.

**Common categories:**
- Fantasy
- Science Fiction
- Romance
- Mystery / Thriller
- Young Adult
- Children's Books
- Non-Fiction
- Biography / Memoir

[Learn about book categories](/taxonomies/book-categories.md)

### Series

Group related books into series with proper ordering. Essential for trilogies, sagas, and ongoing series. AuthorKit automatically maintains reading order.

[Managing book series](/taxonomies/book-series.md)

### Age Ranges

Tag books with appropriate age ranges to help readers find suitable content. Useful for children's books, young adult fiction, and educational materials.

**Standard age ranges:**
- 0-2 (Board Books)
- 3-5 (Picture Books)
- 6-8 (Early Readers)
- 9-12 (Middle Grade)
- 13-18 (Young Adult)
- 18+ (Adult)

[Learn about age ranges](/taxonomies/age-ranges.md)

## Book Limits

### Free Tier
Manage up to **10 books** in the free version of AuthorKit. Perfect for new authors building their catalog.

### Pro Tier
**Unlimited books** with AuthorKit Pro. No restrictions on catalog size, ideal for prolific authors and established writers.

## Bulk Operations

WordPress provides built-in bulk editing capabilities:

1. Go to **Books > All Books**
2. Select multiple books using checkboxes
3. Choose **Bulk Actions** from the dropdown
4. Perform actions like:
   - Edit metadata for multiple books
   - Delete multiple books
   - Change categories or series assignments

## Import and Export

### Exporting Books

Export your complete book catalog for backup or migration:

1. Go to **Tools > Export** in WordPress admin
2. Select **Books** as the content type
3. Click **Download Export File**
4. Receive an XML file with all book data

### Importing Books

Import books from another WordPress site or backup:

1. Go to **Tools > Import**
2. Select **WordPress** importer
3. Upload your export XML file
4. Map authors and import content

## Search and Filter

The All Books screen provides powerful filtering:

- **Search** by title, author, or content
- **Filter by date** (publication or added date)
- **Filter by category** to view specific genres
- **Filter by series** to see series books
- **Filter by status** (Published, Draft, Coming Soon)

## Book Status Options

### Published
The book is live and visible on your website. Available in book archives, shortcodes, and single book pages.

### Draft
The book is saved but not publicly visible. Use for works in progress or books you're not ready to showcase.

### Coming Soon
The book is publicly visible but marked as upcoming. Perfect for pre-orders and building anticipation.

## Quick Edit vs Full Edit

### Quick Edit
Click **Quick Edit** on any book in the All Books list to modify:
- Title
- Status (Published/Draft/Coming Soon)
- Categories
- Series
- Publication date

### Full Edit
Click the book title or **Edit** to access the complete editor with all 27+ metadata fields, cover image upload, and full description editing.

Write compelling book descriptions that:

- Avoid spoilers
- End with a call-to-action

## Troubleshooting

**Can't find Books menu:**
- Ensure AuthorKit plugin is activated
- Check user role has book management permissions
- Try refreshing the admin page

**Changes don't appear on site:**
- Click **Update** or **Publish** to save
- Clear browser cache
- Clear WordPress cache if using a caching plugin
- Check book status is set to "Published"

**Reached book limit:**
- Free tier is limited to 10 books
- Upgrade to AuthorKit Pro for unlimited books
- Delete draft books to free up slots

**Book not showing in shortcodes:**
- Verify book status is "Published"
- Check category/series assignments match shortcode parameters
- Ensure shortcode spelling is correct
