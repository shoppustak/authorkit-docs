# Book Cover Images

## Why Book Covers Matter

Your book cover is the first thing readers see. It's your book's visual identity, primary marketing tool, and often the deciding factor in whether someone clicks to learn more. A professional cover builds credibility and trust.

## Technical Specifications

### Image Dimensions

**Minimum Width:** 800 pixels
**Recommended Width:** 1000-1500 pixels
**Maximum Width:** 2500 pixels (for print quality)

**Aspect Ratio:** 2:3 or 6:9 (standard book proportions)
- Width: Height ratio of approximately 0.67
- Example dimensions: 1000px × 1500px

### File Format

**Best Choice:** JPG (JPEG)
- Excellent compression
- Smaller file sizes
- Best for photographs and complex graphics
- Universally supported

**Alternative:** PNG
- Lossless compression
- Larger file sizes
- Best for text-heavy covers
- Supports transparency (though not needed for covers)

**Avoid:** GIF
- Limited color palette
- Poor quality for covers
- Larger file sizes than JPG

### File Size

**Target:** Under 500KB
**Maximum:** 1MB
**Minimum:** 100KB (ensure quality isn't too compressed)

Smaller files load faster, improving user experience and SEO. Most modern image editors can optimize JPGs to achieve great quality under 500KB.

## Design Best Practices

### Professional Design

**Invest in Quality:**
- Hire a professional book cover designer
- Use pre-made templates from BookBrush, Canva, or Reedsy
- Match genre conventions for your category
- Ensure design looks good at thumbnail size

### Typography

**Text Legibility:**
- Use readable fonts, even at small sizes
- High contrast between text and background
- Author name should be visible but not overwhelm
- Subtitle should be legible at medium sizes

**Test at Multiple Sizes:**
- Thumbnail (100px wide): Title must be readable
- Medium (300px wide): All text should be clear
- Full size: Fine details should be sharp

### Color and Contrast

**High Contrast:** Covers with strong contrast stand out in search results and catalogs.

**Genre Conventions:**
- Romance: Often vibrant colors, couples, elegant fonts
- Thriller/Mystery: Dark colors, bold typography
- Fantasy: Epic scenes, ornate fonts, magical elements
- Science Fiction: Sleek, modern, technological imagery
- Literary Fiction: Minimalist, artistic, subtle

### Composition

**Focal Point:** One clear focal point (face, object, or text)
**Rule of Thirds:** Position key elements along thirds grid
**Negative Space:** Don't overcrowd - let design breathe
**Alignment:** Keep elements aligned and balanced

## Uploading Your Cover

### Step 1: Prepare the Image

Before uploading:

1. **Resize** to recommended dimensions (1000-1500px wide)
2. **Optimize** file size (target 500KB or less)
3. **Check quality** by zooming in - text should be crisp
4. **Convert** to JPG if needed
5. **Rename** file descriptively: `dragon-quest-book-cover.jpg`

### Step 2: Upload to WordPress

1. Edit your book in WordPress admin
2. Find **Featured Image** metabox (right sidebar)
3. Click **Set featured image**
4. Click **Upload Files** tab
5. Drag and drop your cover image or click **Select Files**
6. Wait for upload to complete
7. Click **Set featured image**

### Step 3: Add Alt Text

After uploading, add alt text for SEO and accessibility:

1. Click the image in Media Library
2. Find **Alt Text** field
3. Enter descriptive text: "Cover of The Dragon's Quest showing dragon over mountain landscape"
4. Click **Save**

Alt text helps search engines understand your image and assists visually impaired readers using screen readers.

## Image Optimization

### Before Uploading

Use free tools to optimize images:

**TinyPNG** (tinypng.com)
- Reduces JPG and PNG file sizes
- Maintains quality
- Simple drag-and-drop interface

**Squoosh** (squoosh.app)
- Google's image optimizer
- Advanced compression options
- Visual quality comparison

**Photoshop / GIMP**
- Save for Web option
- Quality slider (80-90% is usually perfect)
- Preview file size

### WordPress Plugins

Install optimization plugins to automatically compress uploads:

**Smush**
- Free version available
- Optimizes on upload
- Bulk optimize existing images

**ShortPixel**
- Excellent compression
- Preserves quality
- Generous free tier

**Imagify**
- Simple setup
- Multiple quality levels
- Automatic optimization

## Responsive Display

AuthorKit automatically creates multiple image sizes for responsive display:

**Thumbnail:** 150×150px (for admin)
**Medium:** 300×450px (for lists and grids)
**Large:** 683×1024px (for single book pages)
**Full:** Original dimensions (for lightbox/zoom)

WordPress generates these automatically when you upload. Your theme uses the appropriate size for each context.

## Multiple Formats/Editions

If you have different covers for different editions:

### Option 1: Use Primary Edition Cover
Choose your main edition (usually paperback or ebook) and use that cover for the book entry.

### Option 2: Create Separate Book Entries
Create individual book entries for each format:
- "The Dragon's Quest (Hardcover)"
- "The Dragon's Quest (Paperback)"
- "The Dragon's Quest (eBook)"

Each can have its own cover variation.

### Option 3: Image Gallery (Advanced)
Use WordPress's built-in gallery feature to add alternate covers in the book's content area. Readers can view all editions.

## Common Cover Mistakes

### Too Much Text
**Problem:** Title, subtitle, author, review quotes, tagline all competing
**Solution:** Prioritize title and author. Remove or minimize secondary text.

### Low Resolution
**Problem:** Blurry or pixelated when viewed large
**Solution:** Use high-resolution source images, minimum 800px wide

### Poor Thumbnail Legibility
**Problem:** Title unreadable at small sizes
**Solution:** Test at 100px wide. Increase font size or simplify design.

### Generic Stock Photos
**Problem:** Looks unprofessional or cliché
**Solution:** Hire designer or use curated stock photo sites with exclusive licenses

### Wrong Aspect Ratio
**Problem:** Square or wide covers look wrong for books
**Solution:** Use 2:3 ratio (portrait orientation like physical books)

### Overcomplicated Design
**Problem:** Too many elements, busy background, confusing
**Solution:** Simplify. One strong focal point, clean typography

## Cover Updates

You can update your book cover anytime:

1. Edit the book
2. Click **Featured Image**
3. Click **Remove featured image**
4. Upload new cover
5. Click **Update**

WordPress automatically clears cached versions and displays the new cover.

**When to update covers:**
- Rebranding or redesign
- New edition release
- Award stickers added
- Professional redesign
- Format changes (hardcover release)

## Testing Your Cover

### Thumbnail Test
Resize your cover to 100px wide. Can you still read the title? If not, increase title font size or simplify the design.

### Genre Comparison
View your cover alongside bestsellers in your genre. Does it fit? Does it stand out? Does it look professional?

### Multiple Device Test
View your book page on:
- Desktop computer
- Tablet
- Mobile phone
- E-reader

Ensure cover displays properly at all sizes.

### A/B Testing
Create two cover variations and test with readers:
- Ask in reader groups
- Post polls on social media
- Test with email list

## Accessibility

### Alt Text
Always add descriptive alt text for screen readers:
- Describe what's shown: "Fantasy book cover featuring dragon flying over mountains"
- Include title and author: "Cover of The Dragon's Quest by Jane Smith"
- Keep it concise: 1-2 sentences maximum

### Color Contrast
Ensure sufficient contrast for visually impaired readers:
- Text should stand out from background
- Use contrast checker tools
- Minimum 4.5:1 contrast ratio for body text
- Minimum 3:1 for large text

## Copyright and Licensing

### Stock Photos
If using stock photography:
- Purchase commercial license
- Read terms carefully
- Keep license receipts
- Don't use free images with restrictions

### Designer Contracts
When hiring designers:
- Ensure you receive full rights
- Get source files
- Clarify ownership
- Specify usage rights (print, ebook, audiobook, marketing)

### Font Licenses
Cover fonts must be licensed for commercial use. Many free fonts prohibit commercial use. Verify licensing before publishing.
