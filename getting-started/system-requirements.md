## Browser Compatibility

Admin interface supports all modern browsers (Chrome, Firefox, Safari, Edge).

## Hosting Compatibility

AuthorKit works on any standard WordPress hosting. Recommended:
- Shared hosting: SiteGround, Bluehost, Kinsta (starter)
- Managed WordPress: WP Engine, Flywheel, Kinsta
- VPS/Cloud: DigitalOcean, AWS, Google Cloud
